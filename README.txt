Набор скриптов для записи изображения экрана, web-камеры и звука

1. Запуск

Командой launcer.vbs

2. Останов

2.1 если установлен nc, то можно командой stopper.cmd
2.2 если не установлен nc, то через WEB:
    - http://localhost:8088 и
    - http://localhost:8089,
    пароль 123,
    В обоих страницах нужно нажать кнопку проигрывателя "Следующий".

    * Остановку через WEB можно выполнить командой stop-http.cmd
